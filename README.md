# airfn — Fortnite external cheat

Team: airproductions · truly · viru · real · icenagger. Five equals, no lead.

## Layout

```
driver/         truly — physmem R/W driver (manual-mapped)
mapper/         truly — iqvw64e-based kdmapper with PDB-resolved offsets
api/            truly — usermode ↔ driver ABI (rose::, process::)
airfn/          usermode cheat client
  main.cpp
  engine/       viru — UWorld / GNames / GObjects walkers
  offsets/      offsets.h (autogen, 1000+ fields from cheatoffsets)
                offsets_manual.hpp (viru — decrypt keys, RVAs, bone indices)
                offsets_verified.hpp (verbatim mirror of cheatoffsets, ground-truth)
scripts/        gen_offsets.mjs — cheatoffsets.com poller
docs/           DUMPER7.md, UWORLD.md, CHEATOFFSETS.md, ROADMAP.md
.github/        CI (build + scheduled offsets refresh)
physmem.sln     MSBuild solution (driver + mapper + airfn)
CMakeLists.txt  CMake preset — builds airfn client only (no WDK needed)
```

## Build

**Full stack (WDK required)**

```
msbuild physmem.sln /p:Configuration=Release /p:Platform=x64
```

Outputs `roseware_driver.sys`, `mapper.exe`, `airfn.exe` under `output directory/Release/`.

**Client only (no WDK)**

```
cmake --preset msvc-x64-release
cmake --build --preset msvc-x64-release
```

## Run

```
mapper.exe roseware_driver.sys
airfn.exe
```

## Offsets

- `airfn/offsets/offsets.h` — regenerated by `node scripts/gen_offsets.mjs` from https://www.cheatoffsets.com/api/games/fortnite/current. ETag-conditional; no-op if unchanged.
- `airfn/offsets/offsets_manual.hpp` — viru's hand-verified UWorld path, decrypt key, GNames/GObjects RVAs. Survives regen.
- CI polls every 30 min and commits when the dump changes.

## Anti-cheat

Fortnite ships **EAC only** (no BattlEye). Everything we do assumes EAC:
Toolhelp32 blocked → kernel-side PEB walk. Handle stripping before EAC scans.
Dumper-7 never injected as-is (recompile + rename).

## Dev-box setup (icenagger + real)

- **Interception driver** for kernel-mouse: https://github.com/oblitum/Interception. `install-interception.exe /install`, reboot. Verify with `sample.exe`.
- **DX11 SDK** ships with Windows SDK — no separate install.
- **WDK 10.0.26100+** required for driver + mapper builds (not for `airfn.exe` alone — CMake preset skips WDK).
